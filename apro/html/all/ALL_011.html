회원탈퇴

회원 탈퇴하시기 전아래의 사항을 먼저 확인해주세요.

  • 잔여 포인트는 탈퇴와 동시에 소멸됩니다. (환불불가)
  • 충전금 잔액이 남아 있을 시, 탈퇴가 불가합니다.
    회원님의 계좌로 잔액을 모두 옮겨주시기 바랍니다.
  • 시리얼의 제휴상품에 투자하셨다면, 탈퇴 이후 상환내역 등의 문의는 해당 제휴사로 문의하여 주시기 바랍니다. (나의 투자현황에서 제휴사를 확인하실 수 있습니다.)
  • 재가입 시, 투자하신 상품내역은 복구되지 않습니다.
  • 탈퇴 사유를 작성해주시면 더욱 발전하는 시리얼이 되겠습니다.
확인
회원탈퇴 사유 선택
  • 투자상품 부족
  • 읽을거리 부족
  • 시스템 장애
  • 더 이상 사용하지 않음
  • 기타(사유 직접 입력)
